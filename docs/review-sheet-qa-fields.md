# Google Sheet 人工核對表 QA 欄位規格

版本：review-sheet-qa-fields-2026-06-29

來源設定：`config/content-qa-rubric.json`

## 1. 適用範圍

這份規格只定義人工核對表的 QA 欄位與升級條件。

現有 Google Sheet 不得由本地文件直接覆寫；任何欄位變更都需要人工確認與可回滾流程。

## 2. 新增欄位

每一張人工核對表，在原本「核准狀態」與「核對意見／修改文字」後面，新增四欄：

| 欄位 | 用途 | 核准要求 |
|---|---|---|
| QA分數 | 0–9 分，依內容治理九項評分 | 必填，至少 7 分 |
| QA阻塞原因 | 命定語氣、醫療宣稱、無行動建議、只講正面等問題 | 有阻塞就不能核准 |
| 修改優先級 | P0-不可使用、P1-需先修、P2-可後修、P3-可用 | 必填 |
| 核對決策 | 核准、需修改、停用 | 必填 |

## 3. 核准規則

一則內容可以升級為正式內容，必須同時符合：

- `核對決策 = 核准`
- `QA分數 >= 7`
- `QA阻塞原因` 空白
- `修改優先級 = P3-可用` 或人工明確允許
- 來源、頁碼、版本資訊齊全

若只填「核准狀態」但沒有 QA 分數，不得建立 approval manifest。

## 4. QA 分數判斷

總分 9 分，每項 0 或 1 分：

1. 主體性。
2. 非命定。
3. 主題清楚。
4. 成熟與陰影都有。
5. 情境相關。
6. 可行動。
7. 不製造焦慮。
8. 交代不確定性。
9. 讀完知道下一步。
