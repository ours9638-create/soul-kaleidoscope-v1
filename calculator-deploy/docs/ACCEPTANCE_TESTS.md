# 靈魂萬花筒計算器｜驗收清單

## A. 自動回歸測試

Cloudflare 每次組建都必須通過核心測試，失敗時禁止部署。

目前 2.2.0 包含 27 項，涵蓋：

- 國曆轉農曆。
- 1989/05/28、1989/12/28 與 2026/07/06 固定換算。
- 農曆 2026/12/01 跨至國曆 2027/01/08。
- 凌晨 12 點使用 24。
- 國曆年月日時分主數鏈。
- 農曆主命數 `37/10/1`。
- `37/10/1` 靈魂等級為 5級。
- 國／農曆流年、流月、流日。
- 貴人數與第一木馬。
- Bob 案例國／農曆皆尚未過生日。

## D. iPhone Safari 驗收

- 首頁、輸入卡片與所有結果卡片左右對齊。
- 日期與時間欄位可正常開啟 iOS 選擇器。
- 國曆生日變更後自動帶出農曆。
- 計算後快速結果與詳細表格更新。
- 寬表格只在表格容器內左右滑動，整頁不橫向漂移。
- 清除重算後輸入與結果回到空白狀態。
- 複製快速結果與完整結果可貼到備忘錄。
- 頁首顯示正確版本與測試通過數。

## E. PWA 與快取驗收

- Safari 重新整理可取得最新版本。
- 加入主畫面後可開啟。
- 版本更新時 cache name 同步更新。
- 新版部署後不持續顯示舊版號。
- 離線時至少能載入最近一次完整快取的介面與資料。

## F. Cloudflare 驗收

- 生產分支：`calculator-deploy-setup`。
- 根目錄：`calculator-deploy`。
- 組建命令：`npm run build`。
- 部署命令：`npx wrangler deploy`。
- 組建紀錄顯示公式測試通過數。
- 測試失敗時組建必須失敗，不得發布正式版本。
- 正式 Worker 維持 `soul-kaleidoscope-v1`，網址不更換。

## G. 發版規則

每次正式修改至少完成：

1. 更新程式。
2. 新增或更新對應回歸案例。
3. 更新版本號與 PWA cache name。
4. Cloudflare 組建成功。
5. 以至少一組固定案例做手機人工驗收。
6. 記錄變更內容與已知限制。
